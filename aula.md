### basicamente após realizar o link remote do github com o vscode, você faz o código no vscode e vai atualizando o github diretamente pelo terminal, atráves dos comandos
```
 git add *(adiciona tudo)
 git pull origin main(pega tudo no github)
 git commit -m 'você adiciona o comentario aqui sobre o que foi feito'
 git push origin main(envia tudo para o repositorio(github))
```
